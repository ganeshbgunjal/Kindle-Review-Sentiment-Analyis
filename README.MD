### Kindle Review Sentiment Analysis
###  Project Overview

This project builds a Natural Language Processing (NLP) model to automatically classify Amazon Kindle book reviews into:

Positive 

Neutral 

Negative 

The goal is to demonstrate real-world sentiment analysis, model comparison, and deployment suitable for a Data Science portfolio.

### Objectives

Perform text preprocessing on raw review data

Convert text into numerical features using NLP techniques

Train and compare multiple machine learning models

Evaluate performance using accuracy, precision, recall, and F1-score

Deploy a simple Streamlit web application for live predictions

### Dataset

Source: Amazon Kindle Reviews (Kaggle)

Typical columns:

reviewText → full review content

summary → short title of review

rating → numeric score (1–5)
